# 虚拟摄像头控制器

一个用于Windows系统的虚拟摄像头程序，可以将静态图片作为虚拟摄像头输出，支持多种图片格式、可调分辨率和动态帧率控制。

## 功能特性

- ✅ 实时虚拟摄像头输出
- ✅ 支持多种图片格式（JPG/PNG/BMP/GIF）
- ✅ 可调分辨率（640x480 至 1920x1080）
- ✅ 动态帧率控制（1-60 FPS）
- ✅ 易用的图形界面
- ✅ 自动匹配图片分辨率（切换图片时自动调整摄像头分辨率）

## 系统要求

- Windows 10/11
- Python 3.8+
- OBS Studio 26.0+（包含虚拟摄像头功能）

## 安装步骤

### 环境测试（可选）

在安装之前，您可以运行测试脚本验证环境配置：

```bash
python test_setup.py
```

测试脚本会检查：
- Python模块是否正确安装
- 虚拟摄像头模块是否可以正常工作
- GUI模块是否正常
- 示例图片是否存在
- 图片加载功能是否正常

### 1. 安装OBS Studio

首先需要安装OBS Studio以提供虚拟摄像头驱动：

1. 访问 https://obsproject.com/download
2. 下载并安装OBS Studio（Windows版本）
3. 安装完成后，启动OBS Studio并启用虚拟摄像头功能：
   - 点击"工具" -> "虚拟摄像机"
   - 点击"启动"

### 2. 安装Python依赖

**方法一：使用安装脚本（推荐）**

在项目目录下双击运行 `setup.bat`，它会自动完成以下操作：
- 安装所有Python依赖包
- 生成示例测试图片
- 显示OBS Studio安装提示

**方法二：手动安装**

在项目目录下运行：

```bash
pip install -r requirements.txt
```

然后生成示例图片：

```bash
python create_sample_image.py
```

### 3. 运行程序

**方法一：使用启动脚本（推荐）**

在项目目录下双击运行 `run.bat`

**方法二：手动运行**

```bash
python main.py
```

## 使用说明

1. **选择图片**：点击"选择图片"按钮，选择要作为虚拟摄像头输出的图片文件
2. **设置分辨率**：在"分辨率设置"区域选择所需的宽度和高度，然后点击"应用分辨率"
3. **设置帧率**：拖动滑块调整输出帧率（1-60 FPS）
4. **启动虚拟摄像头**：点击"启动虚拟摄像头"按钮
5. **停止虚拟摄像头**：点击"停止虚拟摄像头"按钮

### 自动匹配图片分辨率

勾选"自动匹配图片分辨率"复选框后：
- 系统会自动检测所选图片的实际分辨率
- 切换图片时，如果分辨率不同，摄像头会自动重启并使用新分辨率
- 无需手动调整分辨率设置
- 摄像头会短暂重启（约1-2秒），这是正常行为

### 快速测试

项目已包含三个示例图片，您可以用来快速测试程序：

- `sample_640x480.png` - 640x480 分辨率测试图片
- `sample_1280x720.png` - 1280x720 分辨率测试图片
- `sample_1920x1080.png` - 1920x1080 分辨率测试图片

如需创建更多测试图片，可以运行：

```bash
python create_sample_image.py
```

## 注意事项

- 确保OBS Studio已安装并启用了虚拟摄像头功能
- OBS Studio的虚拟摄像头是全局共享的，同一时间只能有一个程序使用
- 如果虚拟摄像头无法启动，请检查OBS Studio是否正在运行并启用了虚拟摄像头

## 故障排除

### 问题：启动虚拟摄像头失败

**解决方案：**
1. 确保已安装OBS Studio
2. 启动OBS Studio并启用虚拟摄像头功能
3. 关闭其他可能使用虚拟摄像头的应用程序

### 问题：找不到虚拟摄像头

**解决方案：**
1. 在OBS Studio中，点击"工具" -> "虚拟摄像机" -> "启动"
2. 重启本程序
3. 检查设备管理器中是否有"OBS Virtual Camera"设备

### 问题：图片无法加载

**解决方案：**
1. 确认图片格式是否支持（JPG/PNG/BMP/GIF）
2. 检查图片文件是否损坏
3. 尝试使用其他图片文件

### 问题：自动匹配分辨率时摄像头重启失败

**解决方案：**
1. 确保OBS Studio正在运行并启用了虚拟摄像头功能
2. 关闭其他可能使用虚拟摄像头的应用程序
3. 等待几秒后重试（系统会自动重试3次）
4. 如果仍然失败，尝试手动停止摄像头后重新启动

## 技术实现

- **虚拟摄像头**：使用pyvirtualcam库
- **图形界面**：PyQt5
- **图片处理**：OpenCV + Pillow
- **依赖驱动**：OBS Virtual Camera
