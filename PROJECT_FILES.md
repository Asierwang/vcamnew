# 项目文件说明

## 核心程序文件

### main.py
- 主程序入口文件
- 启动图形界面

### virtual_camera.py
- 虚拟摄像头核心模块
- 实现VirtualCamera类，提供虚拟摄像头的基本功能
- 功能：
  - 启动/停止虚拟摄像头（支持自动重试）
  - 加载图片并发送到虚拟摄像头
  - 控制分辨率和帧率
  - 线程安全的帧输出
  - 资源释放延迟保护

### gui.py
- 图形界面模块
- 实现VirtualCameraGUI类，提供用户友好的界面
- 功能：
  - 图片选择器
  - 分辨率设置（640x480 至 1920x1080）
  - 帧率控制滑块（1-60 FPS）
  - 启动/停止控制按钮
  - 实时预览
  - 自动匹配图片分辨率功能

## 配置文件

### requirements.txt
- Python依赖包列表
- 包含：
  - pyvirtualcam >= 0.11.0
  - PyQt5 >= 5.15.0
  - Pillow >= 10.0.0
  - opencv-python >= 4.8.0
  - numpy >= 1.24.0

## 工具脚本

### setup.bat
- Windows批处理安装脚本
- 功能：
  - 检查Python环境
  - 自动安装所有依赖包
  - 生成示例测试图片
  - 显示OBS Studio安装提示

### run.bat
- Windows批处理启动脚本
- 简化程序启动流程

### test_setup.py
- 环境测试脚本
- 验证所有组件是否正常工作
- 测试项目：
  - Python模块导入
  - 虚拟摄像头模块
  - GUI模块
  - 示例图片
  - 图片加载功能

### test_auto_resolution.py
- 自动分辨率功能测试脚本
- 测试获取图片分辨率功能
- 验证不同分辨率图片的识别

### test_auto_resolution_fix.py
- 自动分辨率修复测试脚本
- 测试摄像头运行时切换图片的行为
- 验证自动匹配分辨率功能的各种场景
- 包含以下测试：
  - 切换相同分辨率的图片
  - 切换不同分辨率的图片
  - 启动失败处理
  - 自动匹配分辨率禁用时的行为
  - 摄像头未运行时的行为

### test_folder_list.py
- 文件夹列表功能测试脚本

### create_sample_image.py
- 示例图片生成脚本
- 创建不同分辨率的测试图片
- 支持分辨率：
  - 640x480
  - 1280x720
  - 1920x1080

## 使用流程

1. **首次使用**：
   ```
   setup.bat → test_setup.py → run.bat
   ```

2. **日常使用**：
   ```
   run.bat
   ```

3. **重新生成测试图片**：
   ```
   python create_sample_image.py
   ```

4. **环境检查**：
   ```
   python test_setup.py
   ```
